# Periodic SEO Tasks
#
# Read on each heartbeat tick when seoclaw runs as a daemon.
# Format: - [priority|status] Task description
#   priority: high, medium (default), low
#   status:   active (default), paused, completed
#
# Edit freely — these are starting points for an always-on SEO assistant.

- [high] Check Search Console for pages that dropped in clicks or position since the last check, and flag the biggest losers.
- [medium] Look for newly ranking keywords (positions 5-20) that are close to page one and worth a content refresh.
- [medium] Review tracked competitor domains for new top pages or keyword gains we don't cover yet.
- [low|paused] Summarize this week's keyword opportunities saved in OpenSEO and suggest what to target next.
